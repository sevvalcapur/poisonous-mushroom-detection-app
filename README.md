# Poisonous Mushroom Detection App

A mobile application that detects poisonous mushrooms using an Xception-based deep learning model and React Native.

## Technologies Used
- Python
- TensorFlow
- Keras
- Xception
- React Native
- Deep Learning
- Image Classification

## Features
- Mushroom image classification
- Poisonous mushroom detection
- Xception-based deep learning model
- Mobile application interface
- Model integration with React Native
- User-friendly image upload and prediction flow
